# Story 4.1: Proc Alert Icon, Edit Mode, Session Log & Locale System

**Status:** Done
**Epic:** 4 — Diagnostics & v0.3.0
**Version:** v0.3.0
**Commits:** `48e6914`

## Context

Several capabilities were needed simultaneously for the v0.3.0 milestone: a dedicated proc-alert icon slot that highlights when the current suggestion is a proc opportunity (separate from the border pulse); an Edit Mode integration so the frame anchor appears in WoW's built-in layout editor; a persistent session log for diagnosing wrong-icon bugs without needing a live debugger; a keybind cache to avoid redundant lookups; and a locale system so the addon can eventually support translations.

## Acceptance Criteria

- [x] A proc-alert icon slot appears adjacent to the primary icon when the suggested spell has an active proc glow
- [x] Edit Mode registers the display frame so it can be repositioned via the game's built-in layout editor
- [x] `DLog(tag, msg)` appends timestamped entries to `HekiLightDB.sessionLog` (ring buffer, max 500 entries)
- [x] `/hkl log [N]` prints the last N entries from the session log
- [x] On each `/reload`, the current session moves to `HekiLightDB.lastSessionLog` and a fresh log starts
- [x] `keybindCache` stores resolved keybinds by spellID to avoid redundant `GetBindingKey` calls per frame
- [x] `Locale.lua` provides a `HekiLightLocale` global with identity-fallback metatable; loaded before `HekiLight.lua` via TOC order

## Implementation Notes

- Session log ring buffer: `table.remove(t, 1)` on overflow — O(n) but negligible at 500 entries (see ADR-12)
- Change-detection guards (`lastLogSuggID`, `lastSlotSpellID`) prevent high-frequency events from flooding the buffer
- Edit Mode: `EditModeManagerFrame:RegisterFrame(display)` makes the frame movable in the layout editor; position saved to `HekiLightDB.point/x/y`
- `Locale.lua` added to `HekiLight.toc` before `HekiLight.lua`; `HekiLightLocale` uses `__index = function(t,k) return k end` so untranslated keys fall back to their own key string

## Files Changed

- `HekiLight.lua` — DLog, session log, proc-alert icon slot, Edit Mode registration, keybindCache, RAW_SUGG diagnostic logging
- `Locale.lua` — new file, HekiLightLocale global + ptBR block
- `HekiLight.toc` — added `Locale.lua` to load order
